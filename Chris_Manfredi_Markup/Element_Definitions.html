Chapter 2 And 3 Definitions

By: Chris Manfredi

< h1 >
Used for the main headings of a page
< h2, h3, h4, h5, h6 >
Used for sub headings
< p >
Used to create paragraphs for the page
< b >
Used to bold text
< i >
Uses to make text italic
< ol >
Used to show a ordered-list
< li >
Used to showlist items
< ul >
Used to show an unordered-list
< li >
Used to show listed items
< dl >
Used when you have a list of definitions
< dt >
Used when you have a list of defined items
< dd >
Used for a description list
< sup >
Used to super script text
< sub >
Used to sub script text
< br / >
Used to break text to the next line
< hr / >
Used to break the text and create a line
< strong >
Used when you want something to stick out
< em >
Used when you want to add emphasis on the word
< blockquote >
Used when you want a quote but hide where you got it from
< q >
Used when you want to put quotes inside a paragraph
< abbr >
Used when you want to show the box that shows the full meaning when hovering over
< cite >
Used when you want to cite a website you got information from
< dfn >
Used to define a word in the paragraph.
< address >
Used to add an address, phone number, email
<ins >
Used to show text that has been hidden in the page
< del >
Used to show text that was inserted into the page.
< s >
Used to show something that has no value but not delete it
